# README do Projeto Next.js

Este é o README para o projeto Next.js. Ele contém informações sobre como rodar a aplicação na porta 3000 e dá uma dica sobre como salvar um item através da rota de produto.

## Rodando o Projeto

Siga os passos abaixo para rodar o projeto na sua máquina:

1. Clone o repositório para o seu computador:

```bash
git clone https://github.com/LyoDekken/dio-joia
```

2. Navegue até o diretório do projeto:

```bash
cd seu-projeto
```

3. Instale as dependências do projeto:

```bash
npm install
# ou
yarn install
```

4. Inicie a aplicação:

```bash
npm run dev
# ou
yarn dev
```

Agora, a aplicação estará rodando na porta 3000. Você pode acessá-la em 
`[https://dio-joia.vercel.app/]`.

## Salvando um Item na Rota de Produto

Para salvar um item na rota de produto, siga estas etapas:

1. Abra o navegador e vá para `[https://dio-joia.vercel.app/]`.

2. Navegue até a página de produtos.

3. Encontre o item que deseja salvar.

4. Clique no item para abrir os detalhes do produto.

5. Na página de detalhes do produto, você encontrará uma opção para salvar ou adicionar o item ao seu carrinho

Espero que esta informação seja útil para você. Se você tiver alguma dúvida ou encontrar algum problema, não hesite em entrar em contato .
